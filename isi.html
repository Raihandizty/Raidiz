A.pengertian hosting

  • Shared Hosting: Shared hosting adalah layanan hosting di mana banyak pengguna berbagi sumber daya dalam satu server. Setiap pengguna memiliki ruang sendiri, tetapi sumber daya server digunakan bersama. Jika salah satu pengguna menggunakan sumber daya berlebihan, pengguna lain bisa terkena dampaknya. Hosting memberikan ruang penyimpanan, domain, email, dan akses membuat website yang bisa diakses pengunjung.
  • VPS Hosting: Server virtual dengan kontrol penuh.
  • Dedicated Hosting: Server khusus pengguna tunggal.
  • Cloud Hosting: Hosting dengan banyak server terhubung.
